How this guidance is run

This tool compares the current model forecast to past weather patterns over the LIX area, then checks what severe weather happened during the most similar past setups.

  1. Build the history library. Past ERA5 weather fields are paired with local tornado, hail, wind, and significant severe reports.
  2. Download the forecast. The latest RRFS or REFS files are downloaded or reused from the local cache.
  3. Compare each grid cell. For each valid time, the forecast is put on the same grid and scale as the historical library.
  4. Find the closest analogs. The system looks for the past weather patterns that are most similar to the forecast at each grid point.
  5. Convert analogs to probabilities. If 4 of the 40 closest analogs had damaging wind nearby, the wind probability is 10%.
  6. Display the results. The data are exported to GeoJSON and JSON so this page can run in a browser without a backend.
Distance values: lower distance means the forecast looks more like the historical analogs for the fields currently included. Compare distances within the same run, not across different versions of the library.
Important: These are nearest-analog frequencies, not calibrated official NWS probabilities. Use this as experimental situational awareness with radar, observations, SPC guidance, CAMs, and local forecast reasoning.
